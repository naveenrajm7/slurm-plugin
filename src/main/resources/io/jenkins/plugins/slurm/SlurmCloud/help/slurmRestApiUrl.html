The URL of your SLURM REST API server (slurmrestd). This allows Jenkins to communicate with SLURM for job submission, monitoring, and cancellation.

Example formats:
  • http://slurm-controller:6820 (default port)
  • https://slurm-api.example.com:6820
  • http://192.168.1.100:6820

The default port for slurmrestd is 6820, but your installation may use a different port.
