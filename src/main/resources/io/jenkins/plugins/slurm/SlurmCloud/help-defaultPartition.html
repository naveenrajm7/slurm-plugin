Default Slurm partition to submit jobs to when no specific partition is requested in job templates.

This is optional - if not specified, Slurm will use the cluster's default partition. You can override this setting in individual job templates.


Common partition names:
  • compute - General compute partition
  • gpu - GPU-enabled nodes
  • highmem - High-memory nodes
  • debug - Quick testing partition

Check with your Slurm administrator for available partitions in your cluster.
