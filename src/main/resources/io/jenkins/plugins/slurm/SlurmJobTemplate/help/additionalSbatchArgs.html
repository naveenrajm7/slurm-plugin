Additional arguments to pass to the sbatch command when submitting jobs.

Examples:
  • --gres=gpu:1 - Request 1 GPU
  • --constraint=intel - Request Intel nodes
  • --exclusive - Request exclusive access to nodes
  • --mail-type=END --mail-user=user@example.com - Email notifications

Note: Do not include arguments that are already handled by other template fields (like --partition, --nodes, --cpus-per-task, etc.)
