SLURM Job Templates define the parameters for submitting jobs to your SLURM cluster. Each template specifies resource requirements, partition settings, and job configuration that will be used when Jenkins provisions agents for builds.

Use Cases:

  • Light builds: Small jobs with minimal CPU/memory requirements
  • Heavy builds: Resource-intensive jobs requiring multiple CPUs or large memory
  • GPU builds: Jobs requiring GPU resources
  • Specialized partitions: Jobs that need specific hardware or queue priorities

Templates are matched to build jobs based on their labels. If no matching template is found, a default template using the cloud's default partition will be used.
